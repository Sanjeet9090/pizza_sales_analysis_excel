# pizza_sales_analysis_excel
# Project Overview

This project focuses on analyzing pizza sales data using Microsoft Excel to uncover business insights that can help improve decision-making. The dashboard is entirely built using Pivot Tables, Pivot Charts, and Slicers to visualize trends and patterns in customer orders and revenue.

The goal of this project is to understand which pizzas sell the most, which sizes generate more revenue, how customer demand varies month-wise and time-wise, and overall sales performance.
# Tools & Techniques Used
Excel	Data cleaning, transformation & visualization
Pivot Table	Data summarization & KPI calculation
Pivot Charts	Visual representation of major insights
Slicers	Interactive filtering based on Months & Time Intervals
# Key Insights from the Dashboard
Large-size pizzas contributed the highest portion of revenue

Classic & Supreme categories are the best-performing categories

Afternoon time slot shows maximum orders and revenue

Most Selling pizzas include "The Classic Deluxe", "The Sicilian", and "The Barbecue Chicken Pizza"

Monthly revenue follows a steady upward trend with peaks in mid-year
# Conclusion
The dashboard provides clear insights into:

Customer buying patterns

Product performance based on size & category

Peak order timings and revenue-generating periods

Most & least performing pizzas
